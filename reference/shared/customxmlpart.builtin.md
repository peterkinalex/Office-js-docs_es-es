
# <a name="customxmlpart.builtin-property"></a>Propiedad CustomXmlPart.builtIn
Obtiene un valor que indica si el objeto CustomXMLPart se encuentra integrado.

|||
|:-----|:-----|
|**Hosts:**|Word|
|**Disponible en el [conjunto de requisitos](../../docs/overview/specify-office-hosts-and-api-requirements.md)**|CustomXmlParts|
|**Modificado por última vez en**|1.1|

```
var isBuiltIn = customXmlPart.builtIn;
```


## <a name="return-value"></a>Valor devuelto

 **true** si el elemento XML personalizado está integrado. De lo contrario, devuelve **false**.


## <a name="example"></a>Ejemplo




```js
function showXMLPartBuiltIn() {
    Office.context.document.customXmlParts.getByIdAsync("{3BC85265-09D6-4205-B665-8EB239A8B9A1}", function (result) {
        var xmlPart = result.value;
        write(xmlPart.builtIn);
    });
}

// Function that writes to a div with id='message' on the page.
function write(message){
    document.getElementById('message').innerText += message; 
}
```

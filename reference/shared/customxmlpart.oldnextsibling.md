
# <a name="nodedeletedeventargs.oldnextsibling-property"></a>Propiedad NodeDeletedEventArgs.oldNextSibling
Obtiene el siguiente elemento del mismo nivel antiguo del nodo que se acaba de suprimir del objeto **CustomXMLPart**.

|||
|:-----|:-----|
|**Hosts:**|Word|
|**Disponible en el [conjunto de requisitos](../../docs/overview/specify-office-hosts-and-api-requirements.md)**|CustomXmlParts|
|**Agregado en**|1.1|

```
var myNode = eventArgsObj.oldNextSibling;
```


## <a name="return-value"></a>Valor devuelto

Un elemento [CustomXmlNode](../../reference/shared/customxmlnode.customxmlnode.md) que representa el siguiente elemento del mismo nivel anterior del nodo que se acaba de suprimir.


## <a name="support-details"></a>Detalles de compatibilidad

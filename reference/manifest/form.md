
# Elemento Form
Configuración UX para los formularios que su complemento de correo usa cuando se ejecuta en un dispositivo concreto (escritorio, tableta o teléfono).

 **Tipo de complemento:** correo


## Sintaxis:


```XML
<Form>
   ...
</Form>
```
